Coffee
Colors

npm install coffee-colors

Create a color var color= new Color("rgb(0, 159, 255)");
Get a value from it color.to('hex');// => "#009fff"
The supported color formats are rgb, hex, cmyk, hsv and hsl. color.to('hsv');//=> {h:203,s:100,v:100}
 An optional parameter, which is false by default, allows you to transform the color into a CSS-styled string. color.to('hsv', true)//=> "hsv(203,100,100)";
You can also get colors relative to yours. color.complementary()//=> {r:255,g:98,b:0} And can even be used to generate a color scheme. color.triad()//=>
[{r:255,g:0,b:157},{r:157,g:255,b:0}]
